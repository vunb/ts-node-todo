# ts-node-todo

`ToDo App` written in NodeJS with TypeScript

Start app in development mode:

> npm run dev

Start app in production mode

> npm start

# Technology Stack

* Node `v10.15.0`
* Npm `6.4.1`
* Express `^4.16.4`
* Knex / MySQL `^0.16.3`
* Passport-jwt `^4.0.0`
* winston `^3.2.1`
